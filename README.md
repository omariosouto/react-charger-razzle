# Razzle React Loadable Example

## Como usar:
- Clone o projeto, rode um `npm install`;

## Comandos:
- Desenvolvimento: "npm run start:dev"
- Produção em desenvolvimento: "npm run start:prod"
- Produção: "npm run start"
- Build: "npm run build"
